![项目官网](https://class-monthly.pages.dev/)
![项目官网](https://shumaoyong-lele.github.io/Class-Monthly/)

## 项目简介

这是一个班级月刊的官方网站，包含以下功能：

- 首页展示
- 关于我们页面
- 书籍阅读功能
- 深色/浅色主题切换

## 功能特性

- 📚 在线阅读经典文学作品
- 🌙 支持深色/浅色模式切换
- 📱 响应式设计，适配各种设备
- 🚀 简洁直观的用户界面

## 技术栈

- HTML5
- CSS3
- JavaScript
